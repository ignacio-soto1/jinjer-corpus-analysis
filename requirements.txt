# Core dependencies for jinjer-corpus-analysis
# Tested with Python 3.9 and 3.12
#
# Reproducibility note. The published outputs in `outputs/` were produced with
# the version pins below. Earlier (looser) constraints — `spacy>=3.7.0,<4.0.0`
# without a model pin — admit lemmatizer drift across model releases that
# changes a small number of token-level decisions (most visibly the
# h2_displacement_per100 value of "Colossus" on Wallflowers). The pins below,
# combined with the v3.1 dictionary (see `pipeline/process_corpus_discourse_v3.py`,
# DICT_DISPLACEMENT), reproduce the published numbers. Do not loosen these
# pins without re-running the full pipeline and updating the published outputs.
